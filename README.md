# Power BI Financial Dashboard

This project demonstrates financial dashboarding and business intelligence using Power BI, Excel, and DAX.

## Project Overview
The dashboard analyses financial performance through key business metrics, including revenue trends, profitability, P&L movements, balance sheet figures, and financial ratios.

## Key Features
- Executive-facing KPI dashboard
- P&L analysis
- Balance sheet analysis
- Financial ratio reporting
- Drill-down reporting views
- DAX-based calculations
- Clear visual reporting for business decision-making
